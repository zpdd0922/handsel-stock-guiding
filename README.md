# 赠股指引

  32期赠股指引项目

### Usage

启动本地服务

``` bash
$ npm start 
```

打包编译

``` bash
$ npm run build
```

打包编译 - 调试包

``` bash
$ npm run build:debug
```

***
>`VUE_APP_CONSOLE` 是否显示vscode标识,**生产环境**应为`false`
